# Реконструкция проекта "10 Красивых Кнопок"

## Общая информация
Проект представляет собой простую веб-страницу с 10 интерактивными и красиво оформленными кнопками. Каждая кнопка имеет уникальный стиль и анимацию при наведении и нажатии.

## Структура проекта
```
/workspace/
├── .github/
│   └── workflows/
│       └── static.yml          # GitHub Actions workflow для деплоя на GitHub Pages
├── .git/                       # Git репозиторий
├── .gitignore                  # Файл игнорирования для Git
├── .nojekyll                   # Файл для отключения Jekyll на GitHub Pages
├── README.md                   # Описание проекта
├── index.html                  # Основной HTML файл с кнопками
└── manifest.json               # Манифест для PWA
```

## Описание файлов

### index.html
Основной HTML файл проекта, содержащий:
- HTML структуру страницы
- CSS стили для 10 различных градиентных кнопок
- JavaScript для обработки нажатий кнопок
- Адаптивную верстку с использованием CSS Grid
- Анимации при наведении и нажатии
- Ripple-эффект при нажатии на кнопки

#### Особенности CSS:
- Градиентные фоны для каждой кнопки
- Анимации при наведении (transform: translateY(-5px))
- Ripple-эффект при фокусе
- Адаптивный дизайн (grid-template-columns меняется с 2 на 1 при маленьком экране)
- Плавные переходы (transition: all 0.3s ease)

#### Особенности JavaScript:
- Функция handleClick(buttonNumber) для обработки нажатий
- Визуальная анимация при нажатии (scale(0.95) на короткое время)
- Всплывающее окно с информацией о нажатой кнопке

### README.md
Файл с описанием проекта, включающий:
- Название проекта
- Краткое описание
- Список особенностей
- Инструкцию по использованию

### manifest.json
Манифест веб-приложения (PWA) с:
- Названием и описанием приложения
- Настройками иконки (в формате SVG data URL)
- Цветовыми настройками темы
- Настройками отображения

### .github/workflows/static.yml
GitHub Actions workflow для автоматического деплоя статического контента на GitHub Pages:
- Запускается при пуше в ветку main
- Может быть запущен вручную
- Загружает все содержимое репозитория как артефакт
- Деплоит на GitHub Pages

### .gitignore
Файл настройки игнорирования Git, который содержит текстовое сообщение о том, что ничего не должно игнорироваться, так как проект состоит только из исходных файлов.

### .nojekyll
Пустой файл, который отключает Jekyll при развертывании на GitHub Pages, позволяя использовать кастомные статические файлы.

## Технологии
- HTML5
- CSS3 (градиенты, анимации, grid, flexbox)
- JavaScript (обработка событий, DOM-манипуляции)
- PWA (web app manifest)

## Особенности реализации
1. **Адаптивный дизайн**: сетка изменяется с 2 колонок на 1 при маленьком экране
2. **Интерактивность**: каждая кнопка имеет анимации при наведении и нажатии
3. **Уникальные стили**: каждая из 10 кнопок имеет свой градиентный цвет
4. **Современные эффекты**: ripple-анимация при нажатии
5. **Доступность**: кнопки имеют правильные обработчики событий
6. **PWA готовность**: наличие manifest.json файла
