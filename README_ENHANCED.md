# Реконструированный проект: 10 Красивых Кнопок

Добро пожаловать в реконструированную версию проекта "10 Красивых Кнопок"! 

## Обзор

Этот проект представляет собой интерактивную веб-страницу с 10 уникальными градиентными кнопками, каждая из которых имеет свои анимации и эффекты. Проект был реконструирован с улучшениями функциональности и структуры кода.

## Структура проекта

- `index.html` - оригинальная версия проекта
- `reconstructed.html` - улучшенная версия с дополнительными функциями
- `README.md` - базовое описание проекта
- `RECONSTRUCTION.md` - полный анализ реконструкции
- `DOCUMENTATION.md` - подробная документация
- `PROJECT_OVERVIEW.md` - обзор всей структуры проекта
- `manifest.json` - файл манифеста для PWA
- `start_server.sh` - скрипт для запуска локального сервера

## Новые функции в реконструированной версии

1. **Статистика нажатий** - отслеживание количества нажатий для каждой кнопки
2. **Сохранение в localStorage** - статистика сохраняется между сессиями
3. **Уведомления** - всплывающие сообщения при нажатии кнопок
4. **Улучшенная визуальная обратная связь** - более плавные анимации
5. **Улучшенная адаптивность** - лучше выглядит на разных устройствах
6. **Кнопка сброса статистики** - возможность сбросить счетчики

## Запуск проекта

### Локальный запуск

1. Запустите локальный сервер:
   ```bash
   ./start_server.sh
   ```

2. Откройте в браузере:
   - Оригинальная версия: http://localhost:8000/index.html
   - Реконструированная версия: http://localhost:8000/reconstructed.html

### Прямое открытие в браузере

Вы также можете просто открыть HTML файлы напрямую в браузере:
- `index.html` - для оригинальной версии
- `reconstructed.html` - для улучшенной версии

## Особенности реализации

1. **Адаптивный дизайн**: сетка изменяется с 2 колонок на 1 при маленьком экране
2. **Интерактивность**: каждая кнопка имеет анимации при наведении, нажатии и фокусе
3. **Уникальные стили**: каждая из 10 кнопок имеет свой градиентный цвет
4. **Современные эффекты**: ripple-анимация при фокусе, плавные переходы
5. **Персистентность данных**: статистика сохраняется между сессиями (в реконструированной версии)

## Технологии

- HTML5
- CSS3 (градиенты, анимации, grid, flexbox, медиа-запросы)
- JavaScript (DOM-манипуляции, localStorage, обработчики событий)
- Responsive Design (адаптивная верстка)
- PWA готовность (через manifest.json)
