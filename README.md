# 10 Красивых Кнопок

Этот проект представляет собой простую веб-страницу с 10 интерактивными и красиво оформленными кнопками. Каждая кнопка имеет уникальный стиль и анимацию при наведении и нажатии.

## Особенности

- 10 уникальных градиентных кнопок
- Анимация при наведении и нажатии
- Отзывчивый дизайн (адаптируется под мобильные устройства)
- Современные CSS-эффекты
- Интерактивные уведомления при нажатии кнопок

## Использование

Просто откройте страницу в браузере и нажмите на любую кнопку, чтобы увидеть интерактивный эффект.
